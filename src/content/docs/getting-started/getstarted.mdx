## 1. Install FlowMark

1. Click the "Download" button in the top-right corner.
2. Select the latest build for your platform and download the app.
3. Open the downloaded file and follow the installation steps.

## 2. Select Your Notes Folder

1. Launch FlowMark.
2. Choose the root folder where your Markdown files are stored.
3. FlowMark will load files from the folder and start managing your notes.

## 3. Start Editing

You can immediately start organizing and editing your notes. FlowMark provides a Markdown editor, folder-based management, Front Matter support, auto-save, and file watching to support your workflow.
