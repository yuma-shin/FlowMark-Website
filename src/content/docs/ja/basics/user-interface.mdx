FlowMarkのUIは3つの主要なセクションで構成されています。

- サイドバー : 左側のサイドバーは、フォルダツリーを表示し、ノートの階層的な管理を可能にします。
- ノートリスト : サイドバーの隣には、選択したフォルダ内のMarkdownノートのリストが表示され、簡単にアクセスできます。
- エディタ : 画面の右側には、選択したノートを編集するためのMarkdownエディタが配置されており、リアルタイムでプレビューも可能です。

## 新しいフォルダを作成する

サイドバーの上部にある「新しいフォルダ」ボタンをクリックすると、現在選択されているフォルダ内に新しいフォルダが作成されます。これにより、ノートをさらに細かく整理できます。
フォルダをダブルクリックするとパンくずリストのようにフォルダの階層を移動できます。

![create new folder](@/assets/create-folder.png)

## ノートを作成する

ノートリストの上部にある「新しいノート」ボタンをクリックすると、現在選択されているフォルダ内に新しいMarkdownノートが作成されます。これにより、すぐに新しいノートの編集を開始できます。
特定のフォルダに移動した状態でノートを作成すると、そのフォルダ内にノートが作成されます。
ノートをクリックするとエディタに表示され、編集を開始できます。

![create new note](@/assets/create-note.png)
