Notyraではマークダウン記法で記述します。  
また、Github Alertを利用することも出来ます。

## 見出し

先頭に`#`を付けることで見出しを作成することが出来ます。  
`#`の数によって見出しの大きさを変えられます。

```
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
```

# 見出し1

## 見出し2

### 見出し3

#### 見出し4

##### 見出し5

###### 見出し6

## 改行

改行の前に半角スペース`  `を2つ記述することで改行することが出来ます。

```
hoge
fuga(スペース2つ)
foo
```

hoge
fuga  
foo

## 引用

先頭に`>`を付けることで引用表示にすることが出来ます。

```
> 引用
> 引用
>> 多重引用
```

> 引用  
> 引用
>
> > 多重引用

## コードブロック

バッククオート3つで囲むことでコードブロック表示にすることが出来ます。

```
print("Hello, world!")
```

また、バッククオートの後ろに言語名を記載することでコードハイライトが利用できます。

```python
print("Hello, world!")
```

## リスト

### 箇条書きリスト

ハイフン `-`、プラス `+`、アスタリスク `*`のいずれかを先頭に記載することで箇条書き表示にすることが出来ます。

```
- リスト1
    - リスト1_1
        - リスト1_1_1
        - リスト1_1_2
    - リスト1_2
- リスト2
- リスト3
```

- リスト1
  - リスト1_1
    - リスト1_1_1
    - リスト1_1_2
  - リスト1_2
- リスト2
- リスト3

### 番号付きリスト

`番号.`を先頭に記載することで番号付きリストにすることが出来ます。

```
1. 番号付きリスト1
    1. 番号付きリスト1-1
    1. 番号付きリスト1-2
1. 番号付きリスト2
1. 番号付きリスト3
```

1. 番号付きリスト1
   1. 番号付きリスト1-1
   1. 番号付きリスト1-2
1. 番号付きリスト2
1. 番号付きリスト3

## 表

`-`と`|`を使ってtableを作成することが出来ます。

```
| TH1 | TH2 |
----|----
| TD1 | TD3 |
| TD2 | TD4 |
```

| TH1 | TH2 |
| --- | --- |
| TD1 | TD3 |
| TD2 | TD4 |

```
| 左揃え | 中央揃え | 右揃え |
|:---|:---:|---:|
|1 |2 |3 |
|4 |5 |6 |
```

| 左揃え | 中央揃え | 右揃え |
| :----- | :------: | -----: |
| 1      |    2     |      3 |
| 4      |    5     |      6 |

## Github Alert

先頭に`>`を付け、[!〇〇]と記載するとGithub Alertを利用することが出来ます。

```
> [!NOTE] NOTEです
> NOTE

> [!TIP]
> TIPS

> [!IMPORTANT]
> IMPORTANT

> [!WARNING]
> WARNING

> [!CAUTION]
> CAUTION
```

> [!NOTE] NOTEです
> NOTE

> [!TIP]
> TIPS

> [!IMPORTANT]
> IMPORTANT

> [!WARNING]
> WARNING

> [!CAUTION]
> CAUTION

## 便利機能

テキストをドラッグして選択した際に、上部にツールバーが表示されます。
ツールバーで任意の文字効果を選択することで1クリックで文字効果を追加できます。
